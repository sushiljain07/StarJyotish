# Gemstone Recommendations by Ascendant (Lagna)

This is a quick-reference cross-cut. Use it when the user asks "I'm a [Ascendant] — which stones suit me?" Then drill into the individual stone files for the conditions on each.

The notation conventions:
- **YS** = Yellow Sapphire, **BS** = Blue Sapphire, **HL** = House Lord (the planet ruling that house).
- "I, V, IX" etc. refer to the houses where the stone's ruling planet must sit for the stone to be a strong recommendation.
- "Lifetime" = wearable lifelong if those conditions are met.
- "Dasha only" = recommend only when the ruling planet's mahadasha or antardasha is running.
