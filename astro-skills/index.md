# Astro Skills Index

This index lists all available Vedic astrology and numerology skills. Each entry shows the skill `name` (used for lookup), its file path, and when to use it.

---

## How to Use This Index

Each skill lives at `<category>/<skill-name>/SKILL.md`. Some skills have a `references/` subfolder with supplemental detail files that the skill's SKILL.md can reference for deeper lookups.

**Trigger rule**: Match the user's question topic to the skill category below, then load `SKILL.md` for that topic. Load reference files only when more detail is needed.
